# Moom App Prototype 🌙

A basic real-time messaging application with a built-in admin monitoring engine. Users can create a custom username and password without an email address.

## Features
- Custom user authentication with `bcryptjs` password hashing.
- Real-time communication powered by `Socket.io`.
- Server-side centralized chat monitoring logs.
